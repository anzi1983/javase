使用标识符时，需要遵守几条规则：
 1.  标识符可以由字母、数字、下划线（_）、美元符（$）组成，但不能包含 @、%、空格等其它特殊字符，不能以数字开头。
	譬如：123name 就是不合法滴
 2.  标识符不能是 Java 关键字和保留字（ Java 预留的关键字，以后的升级版本中有可能作为关键字），但可以包含关键字和保留字。
	如：不可以使用 void 作为标识符，但是 Myvoid 可以
 3.  标识符是严格区分大小写的。 所以涅，一定要分清楚 imooc 和 IMooc 是两个不同的标识符哦！
 4.  标识符的命名最好能反映出其作用，做到见名知意。

变量命名：
1、变量名由多单词组成时，第一个单词的首字母小写，其后单词的首字母大写，俗称骆驼式命名法（也称驼峰命名法），如 myAge
2、变量命名时，尽量简短且能清楚的表达变量的作用，做到见名知意。如：定义变量名 stuName 保存“学生姓名”信息

当然自动类型转换是需要满足特定的条件的：
1.  目标类型能与源类型兼容，如 double 型兼容 int 型，但是 char 型不能兼容 int 型
2.  目标类型大于源类型，如 double 类型长度为 8 字节， int 类型为 4 字节，因此 double 类型的变量里直接可以存放 int 类型的数据，但反过来就不可以了

所谓常量，我们可以理解为是一种特殊的变量，它的值被设定后，在程序运行过程中不允许改变。
语法：final 常量名 = 值;
常量名一般使用大写字符

变量：
		JAVA中的变量必须先声明才能够使用
		变量的命名规则：
		1，变量必须用字母，数字，下划线或者$符号组成
		2，不能以数字开头
		3，不能是JAVA中的保留字  for break if
		4，不建议使用汉字
		5，尽量使用有意义的单词

JAVA中输出数据：
System.out.print(变量名)；  输出变量值，但是不换行
System.out.println(变量名)； 输出变量值，换行

变量生命周期：从变量的定义开始，直到该变量所处的大括号结束。
局部变量不能在起生命周期后被使用，因为当变量的生命周期结束后JAVA的内存管理器会自动回收该变量的内存

变量的四则运算：
余数的符号位取决于被除数。


算术运算符赋值时，++ 和 -- 既可以出现在操作数的左边，也可以出现在右边  左边;先自增后赋值  右边：先赋值后自增
自增和自减运算符只能用于操作变量，不能直接用于操作数值或常量


联系：
1，用户输入两个数，然后分别输出这两个数的 + - * /的结果 限定为 short int float double 中任意一个
2，随机生成一个迷宫，然后输出该迷宫，如果该迷宫是通的，则输出路径，如果不通则提示该迷宫不通。（9行9列）
